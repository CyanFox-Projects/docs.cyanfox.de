## Introduction

CyanFox-Poll is a simple Poll-System, where you can create Polls and vote on them.

It is built with
- [Laravel](https://laravel.com)
- [Livewire](https://livewire.laravel.com)
- [Tailwind CSS](https://tailwindcss.com)
- [DaisyUI](https://daisyui.com)
- and more
