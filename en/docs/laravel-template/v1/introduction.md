## Introduction

The Laravel-Template is a template for Laravel projects. It includes a lot of features like:

- **User Profile with**
    - Avatar Upload
    - Theme Settings
    - Session Management
    - Two-Factor Authentication
    - API-Keys
    - Activity Log

- **Admin Panel with**
    - User Management
    - Role Management
    - Permission Management
    - Activity Log
    - Alerts

- **API with Documentation**
- **Nav and Sidebar**
- **Build with**
    - TailwindCSS
    - AlpineJS
    - Livewire 3
    - Laravel 10
    - DaisyUI
    - Lucide Icons
    - MaryUI
    - And many more
- **and much more**
